## 虚拟化技术基础原理

    第49天 【虚拟化技术基础原理（03）】

- CPU 虚拟化
    + 模拟：emulation
    + 虚拟：virtulization
        * 完全虚拟化（full-virtualization）
            - BT：二进制翻译（软件）
            - HVM：硬件辅助的虚拟化（硬件）
        半虚拟化（para-virtualization）

- Memory：
    + 进程：线性地址空间
    + 内核：物理地址空间
    + MMU Virtualization
        * Inter：EPT, Extended Page Table
        * AMD：NTP, Nested Page Table
